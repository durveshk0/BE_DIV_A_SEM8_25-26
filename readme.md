🔍 Project Overview

The GSBG Lead Intelligence System is designed to enhance lead qualification and engagement by combining behavioral analytics with machine learning.

🚀 Key Features
Behavior Tracking
Captures user interactions such as scroll depth, time spent, and engagement patterns using cookies.
Lead Classification (Hybrid Model)
Combines:
Machine Learning (TF-IDF + classification models)
Rule-based scoring system
Dynamic Lead Scoring
Classifies leads into categories (e.g., Hot, Warm, Cold) based on behavior and intent signals.
Automated Engagement
Integrates with automation tools (e.g., n8n) to trigger follow-ups and actions based on lead category.
Privacy-Aware Design
Avoids intrusive data collection while maintaining meaningful insights.
🧠 Tech Approach (High-Level)
Behavioral Data Collection (Frontend Tracking)
Feature Engineering using engagement metrics
Text Vectorization using TF-IDF
Model Training using multiple ML algorithms (ensemble-style selection)
Rule-Based Decision Layer for enhanced accuracy
